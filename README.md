# HR Management System

## Overview

HR Management System is an Information Technology Institute (ITI) Graduation Project developed to streamline human resources operations and employee management processes.

The system provides a centralized platform for managing employees, departments, attendance, leave requests, payroll-related workflows, and administrative tasks.

---

## Team Project

This project was developed as part of the ITI Professional Web Development Program.

I contributed to the design, development, and implementation of multiple system modules, backend services, database operations, and business workflows.

---

## Features

* Employee Management
* Department Management
* Attendance Tracking
* Leave Request Management
* Role-Based Access Control
* Authentication & Authorization
* Reporting & Dashboard Features
* Administrative Management Tools
